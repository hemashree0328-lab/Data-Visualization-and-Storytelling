# Data Visualization and Storytelling

## Objective
Create an interactive Power BI dashboard using the Sample Superstore dataset and present key business insights through data visualization.

## Tools Used
- Power BI Desktop
- Sample Superstore Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Quantity KPI
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Sales by State (Map)
- Sales by Sub-Category
- Region and Category Slicers

## Key Insights
- Technology generated the highest sales.
- The West region recorded the highest profit.
- Furniture had lower profitability than other categories.
- Sales varied across months.
- Interactive slicers allow filtering by Region and Category.
